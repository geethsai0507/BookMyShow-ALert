# Code of Conduct

This project is committed to a respectful, safe, and welcoming environment for everyone.

## Our standards

Examples of behavior that helps build a positive community:

- Be respectful and constructive in feedback
- Assume good intent and ask clarifying questions
- Share knowledge and help others learn
- Accept differing viewpoints professionally

Examples of unacceptable behavior:

- Harassment, discrimination, or hate speech
- Personal attacks, insults, or trolling
- Publishing private information without consent
- Spam, intimidation, or disruptive conduct

## Scope

This code of conduct applies in all project spaces, including issues, pull requests, discussions, and other community channels.

## Reporting

If you experience or witness unacceptable behavior, report it by opening a private security report as described in `SECURITY.md`.

## Enforcement

Project maintainers may take any action they consider appropriate, including warning, temporary restriction, or removal from project spaces.

## Good faith

Maintainers will review reports fairly, promptly, and in good faith, with respect for privacy and safety.
